# PulseChat API 入门指南

欢迎使用 **PulseChat API**。本接口由**海豚工作室 (Dolphin Studio)** 提供技术支持，旨在为开发者提供高性能、低延迟的自然语言处理能力。通过本 API，您可以轻松地将尖端的大语言模型（LLM）集成到您的应用、脚本或服务中。

---

## 核心概念

PulseChat 采用 **Chat Completions** 模型。与传统的文本补全不同，它是针对对话设计的，输入是一组包含角色的消息列表。

### 角色说明

| 角色 (Role) | 描述 |
| :--- | :--- |
| **system** | 系统指令，用于设定 AI 的行为、背景或回复风格。 |
| **user** | 用户发送的消息。 |
| **assistant** | AI 的回复内容。可以通过提供历史消息来实现上下文理解。 |
